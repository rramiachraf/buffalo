# Buffalo
Online marketplace where you can buy your favorite games keys.

## Requirements
- Valid Stripe account
- Valid AWS account
- PostgreSQL database
- NodeJS & Yarn

## Stack
The whole project was built using TypeScript.
### Frontend
- React
- Linaria
- Apollo Client 
- Stripe Elements

### Backend
- PostgreSQL
- Apollo Server
- GraphQL Nexus
- AWS S3
- Prisma
- Sharp
- Stripe SDK
