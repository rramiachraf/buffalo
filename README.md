# Buffalo
Online marketplace where you can buy your favorite games keys.

## Requirements
- Valid Stripe account
- Valid AWS account
- PostgreSQL database
- NodeJS & Yarn

## Stack
The whole project was built using TypeScript.
| Frontend | Backend |
| --- | ----------- |
| React | PostgreSQL |
| Linaria | Apollo Server |
| Apollo Client | GraphQL Nexus |
|| AWS S3 |
|| Prisma |
|| Sharp |
|| Stripe SDK |
